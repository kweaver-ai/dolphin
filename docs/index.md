# Dolphin Language SDK 文档

欢迎使用 Dolphin Language SDK！

Dolphin Language SDK 是一个强大的中文AI工作流编排框架，专为构建复杂AI驱动的应用程序而设计。

## 主要特性

- **多智能体系统**: 支持多个AI智能体协同工作
- **领域特定语言 (DSL)**: 简洁直观的编程语言
- **技能系统**: 可扩展的工具和功能模块
- **上下文管理**: 智能的上下文压缩和管理
- **长期记忆**: 持久化记忆存储和检索
- **实验框架**: 完整的基准测试和优化系统
