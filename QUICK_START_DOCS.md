# 文档系统快速开始

## 启动文档服务器

```bash
# 1. 确保依赖已安装
make dev-setup

# 2. 启动文档服务器
make docs-serve
```

## 访问文档

打开浏览器访问：**http://0.0.0.0:30050**

## 主要功能

- ✅ 搜索文档内容
- ✅ 代码高亮和复制
- ✅ 暗色/亮色主题切换
- ✅ 响应式设计（支持手机访问）
- ✅ 实时预览（修改文档后自动刷新）

## 常用命令

```bash
# 启动文档服务器
make docs-serve

# 构建静态文档
make docs-build

# 清理构建产物
make docs-clean
```

## 添加新文档

1. 在 `docs/` 相应目录创建 `.md` 文件
2. 在 `mkdocs.yml` 的 `nav` 部分添加导航链接
3. 重启文档服务器查看更改
